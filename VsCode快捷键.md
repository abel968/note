# VsCode 快捷键
| Press | Function |
| :---: | :---: |
| Ctrl+n | 新建文件 |
| Ctrl+k,v | markdown文件浏览|
| Ctrl+o | 打开文件 |
| Ctrl+` | 打开终端 |
| Ctrl+whitespace | 触发自动补全 |
| Ctrl+alt+'-' | 跳到上一个位置 |
| Ctrl+k, Ctrl+0 | 折叠所有代码块 |
| Ctrl+k, Ctrl+j | 展开所有代码块 |
| Ctrl+/ | 注释当前行 |
| Ctrl+2 | 代码分屏 |
| Ctrl+shift+O | 列出所有变量和函数 |
| Ctrl+b | 隐藏显示左边工具栏 |
